# Right-Word-Game - Simple Bulgarian-English Translation Game

**Right-Word-Game** is a very simple language-learning game.  
Players see a Bulgarian word and must choose the correct English translation from two options. 

<img width="1440" height="696" alt="Screenshot 2025-12-29 at 13 28 21" src="https://github.com/user-attachments/assets/67265cb9-d532-46cc-95ee-d21c7c414ca0" />

The game is built with Node.js and Express for the backend, and vanilla HTML/CSS/JS for the frontend.  
All words are loaded from a JSON file and kept in memory, making the application lightweight and fast.
